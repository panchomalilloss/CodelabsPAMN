## Descripción

Este repositorio contiene varios ejemplos de programas en Kotlin que cubren:

- Impresión de mensajes en consola.
- Uso y manipulación de variables.
- Definición y uso de funciones con parámetros y valores predeterminados.
- Problemas prácticos incluyendo operaciones matemáticas básicas, cadenas y refactorización para evitar código duplicado.

## Ejercicios Incluidos

- Primer programa Kotlin: impresión básica y ordenada.
- Variables: mutables e inmutables, operaciones e impresión formateada.
- Funciones: creación, llamada, parámetros nombrados y por defecto.
- Problemas prácticos: plantillas de cadenas, suma, resta, lógica condicional, y más.

## Ejemplo de Código

fun birthdayGreeting(name: String = "Rover", age: Int): String {
return "Happy Birthday, $name! You are now $age years old!"
}

fun add(number1: Int, number2: Int): Int {
return number1 + number2
}


## Uso

Ejecuta cada función main para observar la salida recomendada. Modifica los valores para experimentar y aprender.
